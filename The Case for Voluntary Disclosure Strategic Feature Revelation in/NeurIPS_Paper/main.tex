\begin{abstract}
This paper addresses a fundamental design question: should a predictive model enforce mandatory feature disclosure or allow voluntary disclosure by participants? Since participants strategically reveal only those features that align with their private objectives, the model must account for opportunistic reporting. For the regression problem, we establish conditions under which the predictive model actually achieves better performance by permitting selective revelation. We extend these results to the classification setting. Finally, we explore a dynamic learning framework where the model interacts repeatedly with participants, presenting an algorithm that adaptively switches between disclosure regimes with established theoretical performance bounds.
\end{abstract}
